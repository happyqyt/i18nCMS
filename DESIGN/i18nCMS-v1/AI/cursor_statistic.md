问题在于下拉框使用了 `absolute` 定位，它是相对于 `relative` 的父容器定位的，而这个父容器在 sticky 的 td 内部。我需要将下拉框改为 `fixed` 定位，并通过 JavaScript 动态计算位置。

完成！我已经做了以下修改：

## 解决方案：

1. **改用 `fixed` 定位**：将下拉框从 `absolute` 改为 `fixed` 定位，这样它会相对于视口定位，不受 sticky 元素的堆叠上下文限制

2. **动态计算位置**：在 `toggleSyncDropdown` 函数中，使用 `getBoundingClientRect()` 获取按钮的实际位置，然后动态设置下拉框的 `top` 和 `right` 位置

3. **添加按钮 ID**：给每个"同步到"按钮添加 ID（`btn-${key}`），方便获取其位置信息

4. **移除 relative 容器**：将包裹按钮的 div 从 `relative` 改为普通的 `inline-block`，因为现在不需要相对定位了

现在下拉框使用固定定位并且 z-index 为 10000，肯定不会被右侧的固定列（z-10）盖住了！
